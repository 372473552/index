在线开发环境 —— Wide

简介

Wide 是一个基于 Web 的团队 Golang IDE，和其他在线 Web 工具一样，Wide 最大的特点就是打开浏览器就可以使用，随时随地。

Wide 是一个开源项目，同时我们也提供了在线服务：https://wide.b3log.org。

特性概览

  • 代码高亮、折叠：Go/HTML/JavaScript/Markdown 等
  • 自动完成：Go/HTML 等
  • 编译检查：编辑器提示编译错误
  • 格式化：Go/HTML/JSON 等
  • 运行：支持同时运行多个程序，方便联调
  • 多用户：团队开发，方便协作
  • 代码导航：跳转到声明，查找使用，文件搜索等
  • Web 开发：前端（HTML/JS/CSS）开发支持
  • go tool：go get/install/fmt 等
  • 项目导入/导出
  • 多主题
  • 调试：基于 gdb 的调试系统（TBD）
  • Git 整合：常用 git 命令可视化操作（TBD）
  • Shell：连接到服务器执行命令（TBD）

到期处理

所有套餐到期后将：

  • 限制新建、编辑、删除文件
  • 限制导入/导出
  • 限制运行

案例

F 餐

特点：

  • 支持导入/导出
  • 可使用未被占用的端口
  • 长期占用资源（网络、CPU、内存等）的运行进程会被 KILL

适用场景：

  • Wide 试用
  • 个人项目临时测试

A 餐

特点：

  • 永久保留数据
  • 长期运行的进程不会被 KILL
  • 使用独立 Wide 的，所有数据、进程都是独立于其他套餐用户的，非常安全
  • 支持 SSH 连接

适用场景：

  • 正式项目
  • 10 人以下团队
  • 非数据库/小流量线上生产服务

B 餐

特点：

  • 更多的内存
  • 支持数据库
  • 支持绑定域名
  • 支持 10 人以上团队
  • 独享 1Mbps 带宽

使用场景：

  • 10-20 人团队
  • 跑线上生产服务（例如企业门户）
  • DevOps：开发、部署、运维一体

独立博客 —— Solo

简介

Solo 是一个成熟、稳定的开源博客。

您可以使用 Solo 来搭建自己的博客，也可以使用我们提供的服务（优势）：

  • 完整的数据导出（SQL）
  • 灵活的域名绑定

在线体验：http://demo.b3log.org，用户名/密码：b3log.solo@gmail.com/b3logsolo

特性概览

  • 文章标签
  • Ping 搜索引擎
  • Atom/RSS
  • sitemap
  • 评论回复及邮件提醒
  • 自定义页面
  • 置顶/相关/随机/站外相关文章
  • 文章、页面永久链接
  • 文章草稿夹、 签名档 、 更新提示
  • 缓存管理
  • 多用户
  • 多语言
  • 多主题
  • 插件
  • MetaWeblog API
  • 数据备份/恢复
