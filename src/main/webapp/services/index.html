现代化社区 Symphony

简介

Symphony （[ˈsɪmfəni]，n.交响乐）是一个现代化的社区平台，因为它：

  • 实现了面向内容讨论的论坛
  • 包含了面向用户分享、交友、游戏的社交网络
  • 集成了聚合独立博客的能力，共建共享优质资源
  • 并且 100% 开源
欢迎到 Sym 官方讨论区了解更多。另外，如果你需要搭建一个企业内网论坛，请使用 SymX。

特性概览

  • 好用的编辑器
  • 智能、灵活的信息架构
  • 满足多样化的发帖需求
  • 人性化的回帖交互
  • 用户个性化设置
  • 编辑历史与匿名发布
  • 对搜索引擎友好
  • 实时的消息通知
  • 好玩的活动
  • 强大的后台管理
  • 开放的内容 API
  • 集成云邮件服务
  • 集成云搜索服务

在线开发环境 Wide

简介

Wide 是一款基于 Web 的团队 Go 语言 IDE，和其他在线 Web 工具一样，Wide 最大的特点就是打开浏览器就可以使用，随时随地。

Wide 可以在本地运行，也可以使用我们提供的在线服务：https://wide.b3log.org。

特性概览

  • 代码高亮、折叠：Go/HTML/JavaScript/Markdown 等
  • 自动完成：Go/HTML 等
  • 编译检查：编辑器提示编译错误
  • 格式化：Go/HTML/JSON 等
  • 运行：支持同时运行多个程序，方便联调
  • 多用户：团队开发，方便协作
  • 代码导航：跳转到声明，查找使用，文件搜索等
  • Web 开发：前端（HTML/JS/CSS）开发支持
  • go tool：go get/install/fmt 等
  • 项目导入/导出
  • 多主题
  • 调试：基于 gdb 的调试系统（TBD）
  • Git 整合：常用 git 命令可视化操作（TBD）
  • Shell：连接到服务器执行命令（TBD）

案例

F 餐
  • 支持导入/导出
  • 可使用未被占用的端口

A 餐
  • 永久保留数据
